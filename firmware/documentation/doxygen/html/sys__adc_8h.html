Internal Temperature Sensor channel identifier.

Note
The ADC channels are defined as follows:
  • ADC_CH_0: ADC0
  • ADC_CH_1: ADC1
  • ADC_CH_2: ADC2
  • ADC_CH_V: VSYS Voltage
  • ADC_CH_T: Internal Temp. Sensor
