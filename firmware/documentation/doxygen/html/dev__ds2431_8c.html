Detailed Description

Driver to perform communication with OneWire device.

Author
Daniel Lockhead
Date
2024

OneWire driver to read / write on the EEprom of the Onewire Device. Plan to be used to detect what's is connected to Interconnect IO Board before turning the main power.

Selects the OneWire device by writing its 64-bit ID to the bus.

This function sends the MATCHROM command followed by the device ID bytes to select a specific OneWire device for subsequent read or write operations.
